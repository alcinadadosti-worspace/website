# Vídeo de Fundo para o Hero

## Como adicionar o vídeo:

1. Acesse um destes sites gratuitos:
   - **Pexels**: https://www.pexels.com/search/videos/silk/
   - **Pixabay**: https://pixabay.com/videos/search/gold%20texture/

2. Busque por vídeos com estas características:
   - **Texturas**: Seda, tecidos fluidos, líquidos metálicos
   - **Cores**: Bege, verde suave, pérola, champagne, dourado suave
   - **Estilo**: Slow motion, abstrato, suave

3. Recomendações específicas:
   - "Gold fabric slow motion"
   - "Silk flowing beige"
   - "Champagne liquid abstract"
   - "Cream fluid texture"

4. Baixe o vídeo em **HD ou Full HD** (não precisa 4K para web)

5. Renomeie o arquivo para: `hero-background.mp4`

6. Coloque nesta pasta: `public/videos/hero-background.mp4`

7. Se o vídeo for muito grande (>10MB), considere:
   - Usar formato WebM (menor que MP4)
   - Comprimir com Handbrake ou similar
   - Objetivo: ~5-8MB para boa performance

## Vídeos Recomendados do Pexels:

- **"A Gold Fabric" (ID: 7430072)** - Tecido dourado fluindo
- Busque: "beige silk", "gold satin", "cream fabric flowing"

## Formatos Suportados:

O código suporta:
- MP4 (melhor compatibilidade)
- WebM (melhor compressão)
- Fallback para imagem estática se necessário
